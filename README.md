<h1>USAGE</h1>
<h3>Press O to open the overlay changer GUI(you can change this later in your controls)</h3>
<h3>Enter your text to obviously, change the watermark</h3>
<h3>(Fun fact, it stores the text in %appdata%\.minecraft\overlaytext.txt so it remembers the text you put in</h3>
<h3>You can enable and disable the overlay using /disableoverlay and /enableoverlay</h3>
